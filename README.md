# CSE Review Studio — Tunet Edition

A mobile-friendly Civil Service Exam practice reviewer built as a simple static web app.  
It includes full-length mock exams, topic drills, caselet activities, weak-spot review, formulas, progress tracking, and dark mode.

Dedicated to **Tunet**.

## Features

- Professional full mock exam
  - 170-item simulation
  - 20 EDQ items
  - 150 scored practice items
- SubProfessional full mock exam
  - 165-item simulation
  - 20 EDQ items
  - 145 scored practice items
- Practice mode with instant correction
- Section drills
- Caselet drills
- Diagnostic mode
- Quick sprint mode
- Weak spot bank
- Progress tracking
- Formula library
- Flashcards
- Dark mode
- Mobile-friendly layout

## Coverage

The reviewer includes practice items for:

- General Information
- Philippine Constitution
- RA 6713
- Human Rights
- Environmental laws and concepts
- English Verbal Ability
- Filipino Verbal Ability
- Numerical Ability
- Analytical Ability
- Data Interpretation
- Clerical Ability
- Data Sufficiency
- Abstract Reasoning

## Important Note

This is a practice reviewer only.

The questions are original practice items based on common Civil Service Exam coverage areas.  
They are not copied from actual Civil Service Commission exam questions and should not be treated as an official CSC reviewer.

Full mock distributions are for practice and pacing only. They are not official CSC item allocations.

## How to Use

Open the site and choose one of the review modes:

1. Use **Practice** for learning with instant explanations.
2. Use **Mocks** for full-length exam simulation.
3. Use **Progress** to check your weak areas.
4. Use **Settings** to personalize your review track.
5. Use **Formulas** and flashcards for quick math review.
