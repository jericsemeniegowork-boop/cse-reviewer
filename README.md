# CSE Review Studio

A focused Civil Service Exam practice reviewer built as a lightweight static web app.

It includes full-length mock exams, topic drills, caselet practice, progress tracking, weak-spot review, and formula cards.

Dedicated to **Tunet**.

## Features

- Professional and SubProfessional mock exam modes
- EDQ-style practice section
- Topic-based practice drills
- Caselet and diagnostic activities
- Weak spot review
- Progress tracking
- Formula library and flashcards
- Dark mode
- Mobile-friendly layout

## Coverage

- General Information
- Philippine Constitution
- RA 6713
- Human Rights
- Environmental laws and concepts
- English Verbal Ability
- Filipino Verbal Ability
- Numerical Ability
- Analytical Ability
- Data Interpretation
- Clerical Ability
- Data Sufficiency
- Abstract Reasoning

## Disclaimer

This is an independent practice reviewer.

All questions are original practice items based on common Civil Service Exam coverage areas. They are not copied from actual Civil Service Commission exam questions and should not be considered official CSC material.

Mock exam distributions are for practice and pacing only.
